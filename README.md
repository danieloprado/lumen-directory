# Lumen Api

[![Run in Postman](https://run.pstmn.io/button.svg)](https://app.getpostman.com/run-collection/6013a4d18a9309c2da53)

See UI project: [React App](https://github.com/danieloprado/react-lumen-app)

## For Developent

```sh
sh ./start.sh --dev
# or
sh ./start.sh -d
```

## For Testing

```sh
sh ./start.sh --test
# or
sh ./start.sh -t
```

## Rebuild docker

```sh
sh ./start.sh [-d|-t] --build
```
